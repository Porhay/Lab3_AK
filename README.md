# Lab3_AK

We have a three parameters:

1. list - optional argument - '-l' or '--list'
2. help - not have argument - '-h' or '--help'
3. value - optional argument - '-v' or '--value'

For example:

Typing `./main -l -h`

Result:
```
1. Argument #1: List
2. Argument #2: Help
```

More example for value argument:

Typing `./main -v150

Result:
```
1. Argument #1 Value=150
```
